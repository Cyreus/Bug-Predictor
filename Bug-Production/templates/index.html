{{cbo}}

Measures the number of classes a given class is dependent on. High CBO indicates greater inter-class dependencies.

Dit:

{{dit}}

Represents the inheritance depth of a class in the hierarchy. A deeper tree implies more inherited behavior and complexity.

Lcom:

{{lcom}}

Measures the cohesion between methods within a class. High LCOM indicates low cohesion and possibly poor class design.
